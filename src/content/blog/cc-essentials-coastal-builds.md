Building coastal in the Sims 4 requires a very specific set of CC if you want things to look genuinely weathered and lived-in rather than catalogue-fresh.

The base game coastal items are fine. They are exactly as fine as they sound — inoffensive, slightly generic, good for a beach house that could exist in any game. For [Tales of Berrinmoore](/story/tales-of-berrinmoore), I needed something different. I needed things that looked like they had been in a house for forty years and had opinions about it.

**Furniture and objects**

The single most useful category for me has been aged wood clutter — driftwood bowls, roughened bookshelves, lamps with slightly dented shades. I have been using the Harlow & Hatch collection from a creator I found through the simblr community (I'll link in the credits below once I double-check the permissions on sharing). The textures read as genuinely old without looking broken or dirty, which is a difficult line to walk.

For kitchen items, I use a lot of cottage-style recolours of base game meshes. If you own Cottage Living, the wooden kitchenware in that pack is already quite good — mostly I need to recolour it to less saturated tones. The bright painted finishes don't read as salt-worn.

**Windows and architectural elements**

This is where CC becomes genuinely essential, because the base game does not have bay windows that look like they belong in a Victorian coastal cottage. I use a set of gridded sash windows that I have had in my mods folder for so long I can no longer remember where they came from — a common and humbling problem. If you recognise them from the screenshots, please let me know.

**Lighting**

Candles, always. Warm Edison bulbs in pendants if I can find a shape I like. I have become very particular about pendant lights. The base game ones have this slightly corporate look that I cannot shake no matter what I do with them, so CC pendants have become non-negotiable for storytelling builds.

More detailed credits and links in the next post — I want to make sure I have everything correctly attributed before I share a full list. In the meantime, [the first chapter](/story/tales-of-berrinmoore/chapter/1) shows most of these pieces in the Berrinmoore house if you want to see how they read together.
